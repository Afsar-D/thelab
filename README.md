# Afsar D — Minimalist Developer Portfolio

A premium, interactive Single-Page Application (SPA) developer portfolio showcasing personal details, skills, works, and professional timeline. Built with vanilla HTML, modern CSS variables, dynamic particle background physics, continuous background relaxing music, and scroll-driven parallax transitions.

## 🚀 Features

* **SPA Route Transitions**: Instant, smooth tab views and URL hash routing for continuous navigation without page reloads.
* **Continuous Background Music**: Playback never resets or stutters when changing pages.
* **Modern percentage Loader Screen**: A digital counter and line-progress indicator tracking simulated page load speeds.
* **Mouse Glow Elements**: technical skill cards and timeline blocks contain canvas-style cursor coordinates glowing borders.
* **Text Scramble Animation**: Dynamic cyclic scrambling developer titles on landing page.
* **Responsive Layouts**: Fully responsive grid systems tailored for desktops and screen viewports down to `320px` width.

## 🛠️ Tech Stack

* **Structure**: Semantic HTML5
* **Styling**: Modern CSS3 (CSS Variables, Flexbox, CSS Grid, Media Queries)
* **Animations**: Pure JS Event Coordinates, Intersection Observer, and CSS Transitions
* **CDNs**: Vercel & Edge Networking

## 📦 Deployment

Deploys automatically via Vercel GitHub hooks. Simply commit your modifications and Vercel takes care of the Edge CDNs distribution.
